Grocery Store V1 - Basic Online Grocery Shopping

Description

This project is a basic online grocery store application where users can browse products, add them to their cart, and proceed to checkout. The project also includes a simple admin interface for adding new products to the store.

Technologies

HTML, CSS, JavaScript, PHP, MySQL

Key Features

  • User registration and login system
  • Product browsing and cart management
  • Checkout and order placement
  • Admin panel for managing products
  • Basic product filtering and sorting
